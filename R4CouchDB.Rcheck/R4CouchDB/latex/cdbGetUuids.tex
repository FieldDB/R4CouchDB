\HeaderA{cdbGetUuids}{Function for id generation}{cdbGetUuids}
\keyword{misc}{cdbGetUuids}
\begin{Description}\relax
Function returns a 128bit uuid requested from CouchDB
\end{Description}
\begin{Arguments}
\begin{ldescription}
\item[\code{cdb}] Only the  connection settings \code{cdb\$port} and
\code{cdb\$serverName} is needed.

\end{ldescription}
\end{Arguments}
\begin{Details}\relax
Simple CouchDB API end point to http://serverName:port/\_uuids. The
number N of ids received from a CouchDB can be increased by
\code{cdb\$queryParam <- paste("count=", N, sep="")}.
For the moment the functions using ids can handle only N=1!
\end{Details}
\begin{Value}
\begin{ldescription}
\item[\code{cdb }] The result of the request is stored in \code{cdb\$id} after
converting the answer into a list using \code{fromJSON()}.

\end{ldescription}
\end{Value}
