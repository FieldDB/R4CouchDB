cdbAddDoc {R4CouchDB}R Documentation

Generates a new document

Description

This function adds a new document to an already existing database

Usage

cdbAddDoc(cdb)

Arguments

cdb The list cdb only has to contain a cdb$dataList which is not an empty list().

Details

This function is called addDoc (which means add a new document). Therefore the cdb$id is requested fresh for every document to add.

Value

cdb The result is stored in cdb$res
