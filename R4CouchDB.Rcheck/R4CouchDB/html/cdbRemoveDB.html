cdbRemoveDB {R4CouchDB}R Documentation

Function to remove a database.

Description

Removing a database means sending a http- "DELETE"- request to http://cdb$serverName:cdb$port/ ...

Usage

cdbRemoveDB(cdb)

Arguments

cdb The cdb has to provide cdb$serverName, cdb$port and cdb$DBName

Details

In cdb maybe a entry cdb$delDBName should be provided for more explicit deleting respectively more secure removing.

Value

cdb The CouchDB answer is stored in cdb$res. Any problems on the R side are reportet in cdb$error
