cdbGetDoc {R4CouchDB}R Documentation

Get a doc from CouchDB

Description

With a given cdb$id this function requests the document.

Usage

cdbGetDoc(cdb)

Arguments

cdb Beside cdb$serverName, cdb$port and cdb$DBName the cdb$id must be given. R errors are reported
in cdb$errors

Value

cdb The result of the request is stored in cdb$res after converting the answer into a list using fromJSON(). If a list entry needed in cdb is not provided cdb$error gives some information.
