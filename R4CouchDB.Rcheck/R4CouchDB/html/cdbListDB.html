cdbListDB {R4CouchDB}R Documentation

Returns all databases

Description

Gives a list of all databases

Usage

cdbListDB(cdb)

Arguments

cdb Only the connection settings cdb$port and cdb$serverName is needed.

Details

Simple CouchDB API end point: http://localhost:5984/_all_dbs

Value

cdb The result of the request is stored in cdb$res after converting the json answer into a list using fromJSON().

Author(s)

wact.b.prot thsteinbock@web.de

References

http://www.omegahat.org/RCurl/ http://www.omegahat.org/RJSONIO/ http://couchdb.apache.org/

See Also

cdbMakeDB

Examples

cdb <- cdbIni()
cdbListDB(cdb)$res
